# 제주 한국재림교회 산티아고 순례길

한국재림교회사에 기록된 제주 유적지를 따라, 그 시절 성도들이 걸어갔을 길을
다음 세대와 함께 걷는 순례길 기획 프로젝트.

## 현재 결과물 (1단계)

| 파일 | 설명 |
|------|------|
| `index.html` | **홈(랜딩)** — 소개·1951 타임라인·코스 한눈에·증언·스탬프·로드맵 |
| `map.html` | 네이버 지도 페이지. **레이어 토글**(🥾 성산 코스 6스톱 / 🗺️ 제주 전역 교회 개요) |
| `story.html` | 코스 소개 + 방문지별 스토리텔링(6스톱) + **어르신 실명 증언** |
| `data/courses.json` | 코스/지점 구조화 데이터 |
| `data/원문기사/` | 재림신문 기획기사 원문(수집본) |
| `자료조사.md` | 제주 재림교회사 사료 정리 + 증언 + 출처 |

> 사이트는 `index.html`(홈)에서 시작 → 상단 네비로 지도·이야기·사료 이동.

## 실행 방법

1. **네이버 지도 API 키 발급**
   - https://www.ncloud.com 가입 → 콘솔 → *AI·NAVER API → Maps* 에서 Application 등록
   - Web 서비스 URL 등록 (예: `http://localhost`, 배포 도메인)
   - 발급된 **Key ID** 복사
2. `index.html` 안의 `YOUR_NCP_KEY_ID` 를 발급받은 키로 교체
3. 브라우저로 `index.html` 열기 (키가 없어도 코스 정보·스토리는 표시됨)

## 1코스 · 전란 중에 핀 꽃 (성산 순례길)

> 1951년 1·4후퇴, 피난선 LST가 성산포항에 닿은 자리에서 시작 →
> 첫 예배·진료소가 세워진 성산초등학교 → 제주 최초 재림 성전 고성리 성산교회

1. 성산포항 — 피난선('아가리배' LST·902명) 상륙지 (1951.01.14)
2. 성산초등학교 — 첫 안식일예배·진료소(장날 600명 무료 진료) (1951.01.20)
3. 성산일출봉·광치기해변 (올레 1·2코스 경유)
4. 오조리 (피난민 정착 마을)
5. 성산교회 (고성리) — 신앙 공동체의 오늘
6. 성산 피난교회 옛 터 — 제주 최초 돌 성전(1951.12.09 헌당), 현재 식당 → 문화유산 등록·되찾기 운동

→ **제주올레 1코스(시흥–광치기)·2코스(광치기–온평)** 와 자연스럽게 겹침.
