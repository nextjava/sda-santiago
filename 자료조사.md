# 제주 한국재림교회 순례길 — 자료조사 노트

> 한국재림교회사·재림마을(재림신문)·디지털제주문화대전·교회 기획기사를 바탕으로 정리한
> 제주 재림교회 역사 사료 모음. 코스 구성의 근거 자료입니다.
> (1차 수집: 2026-06-22)

---

## 1. 한국 재림교회의 시작 (배경)

- **1904년 5월**, 하와이 이민을 신청해 일본 고베항에 머물던 **이응현·손흥조**가
  고베 안식일교회 전도사 **쿠니야 히데**에게 침례를 받아 **한국인 최초의 재림교인**이 됨.
- 손흥조가 귀국하여 한반도에 재림기별을 전함. 일본 미션 책임자 **필드(Field) 목사**가
  한국 최초의 네 교회(**선돌·강대모루·용동·바메기**)를 조직.
- 교세가 서선 → 동해안·남선·중선으로 확대되며 **제주도·울릉도·북간도**에까지 이름.

## 2. 제주 재림교회의 시작 — "전란 중에 핀 꽃" (핵심 스토리)

제주 재림교회의 역사는 **6·25 전쟁 1·4후퇴(1951)** 와 함께 시작된다.

| 날짜 | 사건 |
|------|------|
| 1951.01.11 | 부산항에서 미군 상륙함(LST) 한 척 출항. 한국연합회·서울위생병원 성도 중심 **피난민 902명** 승선 |
| 1951.01.14 밤 9시 | 사흘간 항해 끝에 **성산포항** 도착, 일행 안전 상륙 |
| 1951.01.20 오전 | **성산동국민학교(현 성산초등학교)** 교실에서 **제주 첫 안식일예배** |
| 1951.01.21~ | 학교 교실 한 칸에 **진료소** 개설 → '서울위생병원(현 삼육서울병원) 제주분원'.
  하루 평균 **100~350명**, 장날에는 **최대 600명**의 전염병 환자 **무료 진료** |
| 1951.04 | **고성리**에 예배당 신축 기초공사 시작 |
| 1951.05 | 돌로 건물(성전) 쌓기 |
| 1951.12.09 | **42평 돌 성전 헌당** (제주 최초의 재림 성전 = **성산 피난교회**) |

- **'아가리배'**: 1951년 1월 재림 성도·피난민을 싣고 온 LST 수송선을 부른 **제주도민들의 별칭**.
- 피난민 정착지: **성산포 · 신양리 · 오조리 · 고성리** (성산읍 일대)
- 서울위생병원: 1908년 순안 개원 → 1931년 서울 이전 → 1·4후퇴 때 성산포 피난.
- **성산교회 선교 70주년: 2021년** (→ 2026년 기준 75주년)

### 2-1. 성산 '피난교회' 건물의 현재 — 되찾기 운동 (핵심 스토리텔링 소재)

- 1951년 헌당된 **원래의 돌 성전(피난교회 건물)** 은 현재 **재림교인이 아닌 사람이 식당으로 사용** 중.
  **위치: 성산읍 고성리 327-3**(내러티브 리포트 12편에서 확인 — 그간 위치미상이던 지번).
- 재림교회는 이 건물을 *"제주 복음 역사를 간직한 소중한 유적이자 문화유산"* 으로 보고,
  **문화유산 등록 및 되찾기(회복) 운동**을 추진 중. (재림신문 특별기획 "왜 성산 피난교회를 되찾아야 하는가?")
- → 순례길의 **상징적 종착·기념 지점**이자, 다음 세대에게 전할 "회복의 이야기" 소재.

### 2-2. 피난교회의 매각과 현 성산교회 신축 (내러티브 리포트 3·13편)

- **현재 성산교회(오조리 입구)는 1989년 신축**. 1989년 피난교회 성도가 크게 늘어 예배 공간이 비좁아지자
  예배당 신축을 추진했으나 자금 부족 → **호남합회 지도로 피난교회를 매각**(현 소유주가 1억 원 제시).
  **부복수 집사가 헌납한 오조리 부지** + 매각 대금으로 현 성산교회를 지음. (이후 30여 년간 오조리 입구를 지킴)
- 피난교회 **사용승인일자 = 1951**(읍사무소 확인). 1951년 사진과 돌 하나까지 동일.
- **호남합회 행정위원회 결의**:
  - **호행 88-83**(1988): 성산포 구교회 대지·건물을 **6·25 기념관으로 보존** 결의
  - **호행 89-82**(1989): "현행 도시계획상 철거 불가피"로 위 보존결의 **취소** → 매각으로 이어짐
- 그러나 도시계획은 끝내 시행되지 않아('죽은 계획') 피난교회 건물은 **현존**(주변만 재개발). 현재 식당으로 사용.
- **도등록문화재 등록** 추진의 핵심 난관 = **소유주(비신자) 동의 확보**.
- 피난민 정착 4개 마을(이 기사 표기): **성산포·신양리·고성리·오조리**. 교회 인근 **고성5일장**(끝자리 4·9일).
- 증인: **부복수 집사**(피난교회 부지 헌납·100년사 기록), **한공숙 장로**(마을 토박이), **강관규 수석장로**(오조리 이장).
- 책 표기: 오만규 **『재림교회 100년사』**(=『한국 재림교회 100년사』)의 '성산포 피난생활' 단원.

## 3. 현재 제주 지역 교회 현황

- 한국 첫 교회: **1910년 평안남도 순안**. 교세가 서선 → 전국 → 제주·울릉도·북간도로 확대.
- 제주 지구는 한때 호남합회 소속 → **2001.3 제주교회 운영위원회 구성** → **2009.01.22 연합회 직할선교 지역으로 분리**.
- 현재 **9개 교회 + 2개 예배소**.
- 주요 교회:
  - 제주시: **제주중앙교회**(간월동로 77, 아라2동) · **제주학원교회** · **제주본부교회**
  - 서귀포시: **서귀포교회 · 신서귀포교회**
  - 성산읍: **성산교회**(1951 헌당, 제주 최초)

> 교회 주소·좌표·연락처는 §4-1 표 참조(재림마을 2026 주소록 기준).
> 단, 위 디지털제주문화대전(조성윤 『제주시 50년사』, 2005 기반)의 **제주학원교회·제주본부교회**는
> 현재(2026) 제주대회 목록에 없음 → **개칭·통폐합 추정**(예: 제주학원교회는 **삼육영어학원** 운영 교회).
> 교회별 **설립 연혁(개척 연도)** 은 1차 자료에서 미확인 → '연혁 조사중'.

### 중요 정정·주의 (출처 상충)

- **진료소(제주분원) 위치**: 오만규 교수(역사학자·신양리 출신)는 **동남초등학교(구 서초)** 라 증언.
  반면 마을 어르신 증언·성산교회 70주년 공식기사는 **성산초(구 동초)** 라 함. → **두 설 병기**, 현장 확인 필요.
  (피난민 거처 = 성산초+성산수산고 / 진료소 = 동남초 로 정리하면 모순 없음)
- 침례 인물: **이제명 선교사·김명길 목사**(주례). 위생병원 직원 **한옥선**, 유적지화 자문 **김재호 목사**,
